# 📰 每日热点日报 (Daily Digest)

每天自动抓取全球各大平台热点内容，AI 总结生成日报。

## 数据来源

| 平台 | 类型 |
|------|------|
| 🐦 X/Twitter | 热门话题 |
| 📰 纽约时报 | 国际新闻 |
| 💰 华尔街日报 | 财经新闻 |
| 🇨🇳 FT中文网 | 深度分析 |
| 🇫🇷 RFI法国国际广播 | 国际视角 |
| 🇯🇵 日经中文网 | 亚洲财经 |
| 🌍 路透社 | 全球快讯 |
| 📊 华尔街见闻 | 中国财经 |
| 💻 Hacker News | 科技前沿 |
| 🐙 GitHub Trending | 开源趋势 |

## 在线阅读

访问 GitHub Pages: [https://fulingwei1.github.io/daily-digest/](https://fulingwei1.github.io/daily-digest/)

## 存档

所有日报存放在 `reports/` 目录下，按日期命名：`YYYY-MM-DD.md`

## 自动化

- 每天早上 8:00 (GMT+8) 自动生成
- 由 OpenClaw AI 抓取、总结、发布
- GitHub Actions 自动部署到 Pages
